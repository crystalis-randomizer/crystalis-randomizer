Out of Order

The web tracker is currently out of order.

The best available tracker is currently CodeGorilla's EmoTracker package. It can be found by installing EmoTracker and searching for Crystalis modules (contributions welcome).

If you require a web-based tracker, the best workaround is to use an older version with flags set as similarly as possible. Note that many flags don't have equivalents, so this is a very rough workaround.
